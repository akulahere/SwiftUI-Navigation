# SwiftUI Navigation PoC

Code-first repository for scalable SwiftUI navigation.

## Structure

1. `examples/swiftui-navigation-example/`
Current working application example.

2. `navigation-how-to/starter-code/`
Copy/paste starter architecture with inline comments in Swift files.

## Quick Start

1. Open `examples/swiftui-navigation-example/SwiftUI-Navigation-PoC.xcodeproj`.
2. Run the `SwiftUI-Navigation-PoC` scheme.
3. For a new project bootstrap, copy files from `navigation-how-to/starter-code/Navigation/`.
